# control_ur3sim
Control UR3 simulated robot 

Installation:

    clone repository:

    cd ~/catkin_ws/src

    git clone https://github.com/dominikbelter/control_ur3sim

    compile example:

    cd ~/catkin_ws

    catkin_make

To run:
    
    cd ~/catkin_ws

    rosrun control_ur3sim control_ur3sim
